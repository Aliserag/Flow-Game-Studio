📖 How It Works TECH OVERVIEW

01
HybridCustody

The app creates and manages a Flow account using a sponsored transaction. Your keypair is stored in localStorage. Flow’s HybridCustody contracts let you reclaim full ownership later.

02
NFT Attachments

PowerUp NFTs physically attach to Fighter NFTs using Cadence’s attach keyword. The PowerUp.Boost attachment travels with the Fighter when transferred. Fighter.effectivePower() reads self[PowerUp.Boost] at query time.

03
On-Chain Battle Record

Win/loss counters are mutable fields on the Fighter NFT resource. The BattleArena contract mutates them via a BattleRecorder Cadence entitlement — only authorized code can update combat history.
