📖 How It Works

HybridCustody — The app creates and manages a Flow account for you using a sponsored transaction. Your keypair is stored in localStorage. In production, Flow's HybridCustody contracts let you reclaim full ownership later.

NFT Attachments — PowerUp NFTs physically attach to Fighter NFTs using Cadence's attach keyword. The PowerUp.Boost attachment travels with the Fighter when it's transferred. Fighter.effectivePower() reads self[PowerUp.Boost] to calculate the boosted stat at query time.

On-chain battle record — Win/loss counters are mutable fields on the Fighter NFT resource itself. The BattleArena contract mutates them via a BattleRecorder Cadence entitlement, ensuring only authorized code can update combat history.
