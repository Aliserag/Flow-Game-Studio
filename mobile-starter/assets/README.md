# Assets

Replace these placeholders with your app's actual assets before building.

| File | Required size | Purpose |
|------|--------------|---------|
| `icon.png` | 1024×1024 | iOS + Android app icon |
| `adaptive-icon.png` | 1024×1024 | Android adaptive icon foreground |
| `splash.png` | 1284×2778 | Splash screen |

All images should have transparent backgrounds except `splash.png`.
